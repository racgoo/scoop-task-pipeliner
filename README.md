# scoop-task-pipeliner

Scoop bucket for [task-pipeliner](https://github.com/racgoo/task-pipeliner) - A powerful, condition-based task pipeline runner with beautiful CLI output.

## Installation
```
# Add the bucket
scoop bucket add task-pipeliner https://github.com/racgoo/scoop-task-pipeliner
```

# Install task-pipeliner
```
# install
scoop install task-pipeliner## Updating

# update
scoop update task-pipeliner## Original Repository
```



For more information, documentation, and source code, visit the original repository:

**https://github.com/racgoo/task-pipeliner**
